

 Employee Attrition Analytics (Python + SQL)

This project analyzes employee attrition data stored in a MySQL database and predicts employee churn using Machine Learning.

 Tech Stack-
- Python
- MySQL
- Pandas
- Matplotlib
- Scikit-learn

 Features-
- Stores employee data in MySQL
- Performs attrition analysis by department
- Generates synthetic employee data
- Predicts employee attrition using Logistic Regression

 How to Run-
1. Create database using `db_setup.sql`
2. Install dependencies:
3. generate data
4. run analysis
5. run prediction
